# Browsersync + Gulp.js
### Prerequirement
- installed Node.js 8+

### Install gulpjs globally
```sh
npm install gulp-cli -g
```

### Clone repository

```sh
$ git clone https://github.com/kovinka/gulp-server.git
$ cd gulp-server
$ npm install
```
### Run server
```sh
$ gulp
```
### Open browser
```sh
http://localhost:3000/
```
